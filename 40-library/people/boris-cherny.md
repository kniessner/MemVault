# Boris Cherny

> Software engineer, author, and TypeScript expert. Currently working on Claude Code at Anthropic.

## Bio

Boris Cherny is a software engineer specializing in TypeScript, React, and frontend infrastructure. He is the author of **"Programming TypeScript"** (O'Reilly, 2019), a comprehensive guide to building scalable JavaScript applications with TypeScript.

With 270+ public repositories and 7,500+ GitHub followers, he has contributed significantly to the open-source ecosystem through tools like `json-schema-to-typescript` and `undux`.

Currently, he works on **Claude Code** at Anthropic, building AI-powered developer tools.
