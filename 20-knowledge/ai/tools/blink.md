## Overview

**Blink** (blink.new) is an end-to-end AI application platform founded in 2023. It combines two main products:

1. **Blink AI App Builder** — Vibe coding platform that generates full-stack web and mobile apps from natural language prompts
2. **Blink Claw** — Managed OpenClaw agent hosting (autonomous AI agents with infrastructure)
3. **Blink Cloud** — All-in-one backend infrastructure (DB, auth, storage, hosting, domains, edge functions, queue, AI gateway)

| Attribute | Value |
|-----------|-------|
| **Founded** | 2023 |
| **Type** | AI App Builder + Cloud Infrastructure |
| **Free Tier** | ✅ 10 credits/month |
| **Key Differentiator** | Builds complete apps (frontend + backend + DB + auth + deploy) from chat |
| **Platform** | Web (editor), exports to GitHub |

## What It Builds

- **Websites** — Landing pages, portfolios, blogs, marketing sites
- **Web Apps** — Dashboards, admin panels, productivity tools
- **SaaS** — Subscription-based software with auth, payments, multi-tier plans
- **Mobile Apps** — iOS and Android with React Native / Expo
- **Chrome Extensions** — Browser extensions with live popup preview

## Blink Cloud Infrastructure

Replaces 6–8 separate services with one platform. Included in every plan:

| Capability | Description |
|------------|-------------|
| **Database** | Scalable SQL (Turso) — describe your data, Blink handles setup |
| **Authentication** | Social logins (Google, GitHub, Apple, Microsoft), email/password, magic links, RBAC |
| **Storage** | File upload with CDN, image optimization, progress tracking |
| **AI Gateway** | 200+ models (text, image, video, speech) via OpenAI-compatible API |
| **Edge Functions** | Server-side code close to users — webhooks, secure API calls |
| **Backend** | 1,000+ auto-generated API routes |
| **Queue** | Background tasks, cron jobs, job inspection |
| **Real-time** | WebSocket messaging, presence tracking, instant updates |
| **Email** | Transactional emails with attachments, branding, delivery tracking |
| **Analytics** | Privacy-first pageviews, custom events, UTM attribution, session tracking |
| **Hosting** | Production deploy, global CDN, sub-40ms latency |
| **Custom Domains** | Buy or connect domains, auto SSL |

## Limitations

- Closed platform — code export to GitHub available, but hosting tied to Blink Cloud
- Credit-based AI usage can get expensive for high-volume apps
- Mobile builds done in cloud (not local) — requires internet
- Custom backend logic limited to edge functions
- No self-hosted option for Blink Cloud infrastructure
