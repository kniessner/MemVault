# Factory.ai

> **Agent-native software development.** AI coding agents ("Droids") that automate coding, testing, and deployment across your entire engineering workflow.

## Overview

**Factory.ai** is an enterprise platform that deploys AI coding agents called **Droids** directly into your existing engineering toolchain. Unlike standalone AI editors, Factory meets you where you already work—IDE, terminal, browser, Slack, and CI/CD—without forcing tool switches.

## Droids: Where They Work

Factory's agents integrate across the entire development lifecycle:

| Location | Integration | Use Case |
|----------|-------------|----------|
| **IDE** | VS Code, JetBrains, Vim, Neovim | Inline coding, refactoring, reviews |
| **Web** | Browser dashboard | Non-dev tasks, project management |
| **CLI** | Terminal | Scripting and automation |
| **Slack** | Team chat | Incident response, quick tasks |
| **Linear** | Issue tracker | Auto-assign, implement from tickets |
| **CI/CD** | Build pipelines | Automated code review, self-healing builds |

## Key Features

| Feature | Description |
|---------|-------------|
| **Full-task delegation** | Droids handle complete tasks (refactors, migrations, incident response), not just autocomplete |
| **Parallel Droids** | Spin up many agents simultaneously for large-scale CI/CD, migrations, audits |
| **Self-healing builds** | Agents monitor and fix failing builds automatically |
| **Auto PR creation** | Droids implement solutions from tickets and open PRs with full traceability |
| **Model & tool agnostic** | Works with your existing tools, models, and workflows—no forced switches |
| **Enterprise-grade** | Security, compliance, SAML, SCIM, audit logs |
