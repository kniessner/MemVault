# Browserbase

> Headless browser infrastructure for AI agents and automation.

## Overview

Browserbase provides managed headless browsers for:
- AI agent web browsing
- Web scraping at scale
- Session recording and debugging
- Stealth mode (anti-bot evasion)

## Key Features

- **Managed infrastructure** — No need to run your own Chrome/Puppeteer cluster
- **Session replay** — Record and debug browser sessions
- **Stealth mode** — Bypass common bot detection
- **Integrations** — Works with AI agent frameworks

## Use Cases

- AI agents that need to browse the web
- Automated testing and scraping
- Data extraction from JavaScript-heavy sites
