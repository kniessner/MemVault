## Vibe Coding App Builders

Full-stack app builders that generate code from natural language prompts. "Vibe coding" = describe what you want, AI writes everything.

| Tool | Free Tier | Paid | Self-Host | Type | Best For |
|------|-----------|------|-----------|------|----------|
| **[[bolt.new]]** | ✅ 300K tokens/day, 1M/mo | $25/mo | ✅ Export code | Full-stack web | Rapid prototyping, Next.js/React |
| **[[lovable.dev]]** | ✅ Basic builds | ~$25/mo | ✅ Export code | Full-stack web | Designers, team collaboration |
| **[[v0.dev]]** | ✅ $5/mo credits, 7 msg/day | $30–100/user/mo | ❌ (Vercel lock-in) | Next.js UI | Vercel-deployed React components |
| **[[replit.com]]** | ✅ Daily free agent credits | $20–100/mo | ❌ Cloud only | Full-stack web | Beginners, real-time collab |
| **[[blink]]** | ✅ 10 credits/mo | From $19/mo | ✅ Export to GitHub | Full-stack + mobile | Backend + mobile + Chrome ext |

### Free / Self-Hostable Strategy
All vibe coding tools above let you **export code** and then host it yourself:
1. Build your app in the free tier
2. Export to GitHub / download ZIP
3. Deploy on **free** hosts: Vercel, Netlify, Cloudflare Pages, GitHub Pages, Render
4. For backend: Railway, Render, Fly.io (free tiers)
5. For DB: Supabase free tier, Neon free tier, Turso free tier

**Zero-cost stack:** Build with Bolt free → Export → Deploy on Vercel free + Supabase free = $0

**Recommendation:**
- **Free + best export:** [[bolt.new]]
- **Fastest iteration:** [[lovable.dev]]
- **Vercel-native:** [[v0.dev]] (but locked into Vercel)
- **Beginner-friendly:** [[replit.com]]
- **Mobile + backend:** [[blink]]
