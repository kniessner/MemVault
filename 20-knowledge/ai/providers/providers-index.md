## Provider Details

### [[anthropic|Anthropic]] ⭐
- **Strengths**: Safety, steerability, coding
- **Models**: Claude 3.7/3.5 Sonnet, Haiku
- **Context**: 200K
- **Best for**: Code assistance, reliable outputs

### [[fal-ai|fal.ai]] 🎨
- **Strengths**: 1,000+ generative models, serverless GPU
- **Models**: FLUX, Nano Banana, Kling, Seedance, Veo
- **Pricing**: Pay-per-use (images/videos/seconds)
- **Best for**: Image generation, video generation, fine-tuning

### [[goose-ai|GooseAI]] 💰
- **Strengths**: 70% cost savings, per-request pricing
- **Models**: GPT-Neo, GPT-J, GPT-NeoX, Fairseq (125M-20B)
- **Pricing**: Per-request + output tokens only
- **Best for**: Budget-conscious NLP applications

### [[goose-agent|Goose Agent]] 🤖
- **Strengths**: Open source, MCP extensions, local execution
- **Interfaces**: Desktop app, CLI, API, ACP server
- **Extensions**: 70+ MCP servers
- **Best for**: Privacy-focused agent workflows

### [[groq|Groq]] ⚡
- **Strengths**: Speed, LPU hardware
- **Models**: Llama, Mixtral, Gemma via fast inference
- **Speed**: 800+ tokens/sec
- **Best for**: Real-time applications, low latency

### [[google|Google Gemini]] 🔬
- **Strengths**: 2M context, native multimodal
- **Models**: Gemini 2.0 Pro/Flash, 1.5 Pro/Flash
- **Context**: Up to 2M tokens
- **Best for**: Video analysis, entire codebases

### [[lm-studio|LM Studio]] 💻
- **Strengths**: Beautiful GUI, model discovery
- **Models**: Any GGUF from HuggingFace
- **Type**: Desktop app (Mac/Windows/Linux)
- **Best for**: Non-technical users, comparing models

### [[moonshot-ai|Moonshot AI]] 📖
- **Strengths**: Ultra-long context (2M+)
- **Models**: Kimi K2.5, K1.6
- **Context**: 256K to 2M+
- **Best for**: Book analysis, large documents

### [[ollama|Ollama]] 🦙
- **Strengths**: Simplicity, scriptability
- **Models**: Llama, DeepSeek, Qwen, Mistral
- **Type**: CLI + library
- **Best for**: Developers, automation, scripts

### [[openai|OpenAI]] 🚀
- **Strengths**: Frontier performance, ecosystem
- **Models**: GPT-4o, o3, DALL-E, Whisper
- **Context**: 128K-200K
- **Best for**: General purpose, multimodal apps

### [[openrouter|OpenRouter]] 🔄
- **Strengths**: Provider diversity, fallback
- **Models**: 300+ from multiple providers
- **Type**: API aggregation layer
- **Best for**: Model comparison, redundancy

### [[perplexity|Perplexity]] 🔍
- **Strengths**: Search integration, citations
- **Models**: Sonar, Sonar Pro, Sonar Deep Research
- **Context**: 128K
- **Best for**: Research, grounded answers

### [[z-ai|Z.ai]] 🇨🇳
- **Strengths**: Agentic engineering, Chinese market
- **Models**: GLM-5, GLM-4.7
- **Context**: 128K
- **Best for**: Agent workflows, Chinese language
