# Second Brain and PARA

## Definition

The Second Brain approach treats externalized notes as a working memory system. PARA gives that system a simple top-level organizing frame: Projects, Areas, Resources, Archives.

## Key Points

- organizational systems should reduce friction, not become a second job
- information should be stored where future action or retrieval becomes easier
- simple top-level structure scales better than deep taxonomic complexity

## Why it matters here

This vault is not a pure PARA system, but it shares the same goal: actionable organization with a small set of stable top-level branches.
