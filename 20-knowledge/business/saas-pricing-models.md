# SaaS Pricing Models

## Definition

Pricing is not just monetization. It defines how value is measured, how adoption friction feels, and how revenue scales with customer behavior.

## Key Models

### Seat-based

- easy to understand
- predictable for finance
- weak fit when usage varies more than team size

### Usage-based

- lowers barrier to entry
- ties expansion to actual adoption
- can create billing unpredictability and surprise

### Hybrid

- combines platform fee and variable usage
- often best when there is both baseline product value and bursty consumption

## When to use

- use seat-based when collaboration structure is the main value driver
- use usage-based when compute, messages, or transactions track value directly
- use hybrid when a stable platform plus elastic workloads both matter
